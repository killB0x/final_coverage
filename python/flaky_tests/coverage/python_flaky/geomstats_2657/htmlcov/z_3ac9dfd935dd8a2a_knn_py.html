7 """Classifier implementing the k-nearest neighbors vote on manifolds. 

8 

9 Parameters 

10 ---------- 

11 n_neighbors : int, optional (default = 5) 

12 Number of neighbors to use by default. 

13 weights : string or callable, optional (default = 'uniform') 

14 Weight function used in prediction. Possible values: 

15 - 'uniform' : uniform weights. All points in each neighborhood 

16 are weighted equally. 

17 - 'distance' : weight points by the inverse of their distance. 

18 in this case, closer neighbors of a query point will have a 

19 greater influence than neighbors which are further away. 

20 - [callable] : a user-defined function which accepts an 

21 array of distances, and returns an array of the same shape 

22 containing the weights. 

23 p : integer, optional (default = 2) 

24 Power parameter for the 'minkowski' string distance. 

25 When p = 1, this is equivalent to using manhattan_distance (l1), 

26 and euclidean_distance (l2) for p = 2. 

27 For arbitrary p, minkowski_distance (l_p) is used. 

28 distance : string or callable, optional (default = 'minkowski') 

29 The distance metric to use. 

30 The default distance is minkowski, and with p=2 is equivalent to the 

31 standard Euclidean distance. 

32 See the documentation of the DistanceMetric class in the scikit-learn 

33 library for a list of available distances. 

34 If distance is "precomputed", X is assumed to be a distance matrix and 

35 must be square during fit. 

36 distance_params : dict, optional (default = None) 

37 Additional keyword arguments for the distance function. 

38 n_jobs : int or None, optional (default = None) 

39 The number of parallel jobs to run for neighbors search. 

40 ``None`` means 1; ``-1`` means using all processors. 

41 

42 Attributes 

43 ---------- 

44 classes_ : array, shape=[n_classes,] 

45 Class labels known to the classifier 

46 effective_metric_ : string or callable 

47 The distance metric used. It will be same as the `distance` parameter 

48 or a synonym of it, e.g. 'euclidean' if the `distance` parameter set to 

49 'minkowski' and `p` parameter set to 2. 

50 effective_metric_params_ : dict 

51 Additional keyword arguments for the distance function. 

52 For most distances will be same with `distance_params` parameter, 

53 but may also contain the `p` parameter value if the 

54 `effective_metric_` attribute is set to 'minkowski'. 

55 outputs_2d_ : bool 

56 False when `y`'s shape is (n_samples, ) or (n_samples, 1) during fit 

57 otherwise True. 

58 

59 References
